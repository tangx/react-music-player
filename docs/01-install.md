# 安装项目

```bash
$ npx create-react-app news-system

$ cd my-app
$ yarn start
```


## 安装 sass 支持

```bash
$ yarn add sass sass-loader
```


## 配置网络

```bash
$ yarn add axios
```

**开发环境** 配置跨域代理

> https://create-react-app.dev/docs/proxying-api-requests-in-development

```bash
$ yarn add http-proxy-middleware
```

创建 `src/setupProxy.js`
