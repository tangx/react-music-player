# ant design

> https://ant-design.gitee.io/docs/react/introduce-cn

```bash
$ yarn add antd
```

**使用**： 注意， 需要引入 `antd/dist/antd.css`。

```jsx
import 'antd/dist/antd.min.css';
import { Button } from "antd";


function App() {
  return (
    <div>
      <Button type="primary">Button</Button>
    </div>
  );
}

export default App;
```
